⚖️ Class imbalance — the central problem

Stroke vs no-stroke

Stroke (249) No stroke (4,861)

Why this matters

  • A model that always predicts "no stroke" would score 95.13% accuracy. Accuracy is therefore meaningless here.
  • Real metrics to watch: Recall (caught strokes), Precision (false-alarm rate), and PR-AUC.
  • Only ~249 positive cases — splitting 80/20 leaves ~50 positives in the test set. Single-split metrics will be noisy; stratified K-fold is safer.
  • SMOTE / class_weight are reasonable fixes during training, but never resample the test set.

📈 Age — by far the strongest signal

Stroke rate by age bin (%)

Risk grows roughly exponentially with age. Below 40 it's well under 1%. Above 80 it's more than 1-in-5. Any model is going to lean heavily on this single feature — and SHAP plots in outputs/ confirm it does.

🩸 Glucose — bimodal distribution

Stroke rate by avg_glucose_level (mg/dL)

Glucose distribution in this dataset is bimodal — a normal cluster around 90 and a second hump near 200+ (likely diabetic patients). Once glucose is above ~170, stroke rate triples. Below 170 it's basically flat — meaning a linear model will under-use this feature unless you bin it or add a "high glucose" indicator.

🕳️ The BMI-missing trap (important)

Missingness is NOT random

What this means

  • Patients with missing BMI are ~5× more likely to be stroke positive.
  • Likely they were too sick / hospitalized for BMI to be recorded — i.e. missingness itself is signal.
  • Mean-imputing BMI destroys that signal and even leaks a bit.
  • Better: add a bmi_missing binary flag, then impute. Or use a model that handles NaN natively (XGBoost).

🎯 Key takeaways

What this dataset is good for

  • Practicing imbalanced classification: SMOTE, class weights, threshold tuning, PR curves.
  • Learning feature engineering: BMI missingness as signal, glucose binning, age × condition interactions.
  • Demonstrating explainability (SHAP) — age dominates, easy to verify against medical intuition.

What it's NOT good for

  • Making real clinical claims — only 249 positive cases, single source, no temporal info.
  • Studying gender effects — 1 "Other" row, near-identical Male/Female base rates.
  • Achieving high precision — the prevalence ceiling makes precision >0.5 at decent recall basically impossible.
